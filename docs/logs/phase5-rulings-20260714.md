## 確定内容

- **P5-1**: 未紐付け・評価不能・シナリオ削除済み銘柄の時価は**投資額で代替(コスト評価)+行に「コスト評価」バッジ**
- **P5-2**: 時価 = **EV × 持分(企業価値ベース、netDebt控除なし)+表注記**(現在ネットデット入力は Stage 2 で再設計)
- **P5-3**: 既存データの `investmentDate` は **null 補完**(IRRは「—(投資日未設定)」。createdAt での偽装はしない)
- **P5-4**: 評価基準日は**今日**(UI層で算出、画面とExcelに評価日を明記。エンジンには数値のみ渡す)
- **P5-5**: 並列比較はセクター**混在許可**(共通ブロック+同一セクター2件以上でセクター別ブロック)
- **P5-6**: 比較件数上限 **4件**
- **P5-7**: 医療機器 implied EV/売上マルチプルは **Phase 5 に含めない**(完了後の独立バッチのまま)
- **P5-8**: Excel単票に感度分析(上位10行)・資本政策(ラウンド表+持分推移)を**含める**
- **P5-9**: ファンド集計は**時価総額のみ3点、IRR/MOICはベース基準のみ**

## 補足(記録者注)

- U-23(時価の定義=EVベース)・U-24(未紐付け銘柄=コスト評価)は本裁定で**確定**。実装セッション C1 の engine-spec 追記時に状態を「確定」と記載する。
- これにより Phase 5-B(実装、Sonnet セッション)の着手条件はすべて充足。最初のコミットは C1(engine-spec の Cashflow.t 実数年許容の追記+回帰テスト)。
